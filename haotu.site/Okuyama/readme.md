- 奥山和纱介绍

奥山かずさ（奥山和纱、おくやま かずさ、Okuyama Kazusa），日本女演员，青森县出身，隶属オスカープロモーション旗下。在三沢市被星探发掘为契机开始了模特活动，曾登上周刊Young Jump 2014，2016年在奥克斯宣传举办的「第1回ミス・美しい20代コンテスト」中获得准大奖。出演了2017年1月播出的TBS系列《发现！○○之人》。
